# Jogo da Velha
## Como rodar o projeto?

**Clone o repositório:**
```
git clone https://github.com/Edumorais08/jogo-da-velha.git
```

**Vá para a pasta "jogo-da-velha" mais interna:**
```
cd jogo-da-velha
```

**Instale as dependências e rode o projeto:  =)**
```
npm install
```

```
npm run dev
```
